# MOZKITO Privacy Policy

This public repository only hosts MOZKITO's privacy policy for Google Play Console.

It intentionally contains only minimal static files for the privacy policy page. No MOZKITO app source code, signing files, keystores, passwords, Google credentials, AABs, APKs, or build artifacts are stored here.

Public privacy policy URL after GitHub Pages is enabled:

https://carlosJxD.github.io/mozkito-privacy/
